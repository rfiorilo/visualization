1. Por que apenas números inteiros positivos são suportados?

    A ferramenta foi projetada para trabalhar exclusivamente com valores inteiros positivos.

    Internamente, as comparações são realizadas utilizando representação textual (strings). Para garantir que a ordenação corresponda corretamente à ordem numérica, os inteiros positivos podem ser ajustados para um formato padronizado (por exemplo, com zeros à esquerda).

    Números negativos introduzem o caractere - na comparação, o que altera a lógica de ordenação. Da mesma forma, números reais (por exemplo, 3.14) incluem o caractere ., gerando inconsistências na ordenação textual.

    Como o objetivo da ferramenta é demonstrar o funcionamento das estruturas de dados de forma clara e consistente — e não tratar regras adicionais de interpretação ou conversão numérica — a aplicação aceita apenas inteiros positivos.

  2. Como utilizar a ferramenta em celulares com tela pequena?

    Se estiver utilizando um dispositivo com tela reduzida, ajuste o Canvas Size para dimensões mais adequadas ao seu visor. A aplicação salvará essa preferência em um cookie, de modo que o tamanho escolhido seja mantido nas próximas visitas. Após alterar as dimensões, recarregue a página para aplicar corretamente o novo tamanho.

    Como referência, dispositivos com largura em torno de 500px e altura de 300px costumam funcionar bem para a maioria das estruturas e algoritmos.

    Em dispositivos iOS, é possível definir um tamanho maior que o exibido na tela. Nesse caso, será necessário utilizar zoom ou rolagem para visualizar todo o conteúdo.

  3. A ferramenta utiliza cookies?

    Sim, mas apenas para memorizar suas preferências de largura, altura e velocidade da animação.

    O sistema continuará funcionando normalmente mesmo que você desative os cookies. Nesse caso, apenas não será possível manter automaticamente as configurações de tela e velocidade escolhidas anteriormente.

    Não há qualquer utilização de cookies para rastreamento, publicidade ou coleta de dados externos.

  4. Encontrei um possível erro. Como devo proceder?

    Caso identifique algum erro de tradução ou comportamento inesperado, solicitamos que envie um e-mail descrevendo detalhadamente a ocorrência.

    Inclua, se possível, a sequência exata de operações realizadas, os valores inseridos e o que era esperado como resultado. Quanto mais específicas forem as informações fornecidas, mais eficiente será a análise e eventual correção do problema.

  5. Posso usar estes materiais em minhas aulas?

    Sim! O material pode ser utilizado livremente para fins educacionais.

    Apenas recomendamos que, sempre que possível, sejam criados links para as páginas oficiais, em vez de armazenar o conteúdo localmente. Dessa forma, seus alunos terão acesso à versão mais atualizada da ferramenta.

    Se você utilizar este recurso em suas aulas, ficaremos felizes em receber uma mensagem informando. É sempre interessante saber quem está utilizando o material e em quais contextos educacionais.

  6. Posso usar este material como base para minhas próprias visualizações?

    Sim! Consulte a página de código-fonte para acessar um tutorial introdutório e baixar o pacote com os arquivos necessários.

    Caso utilize o código, solicitamos que mantenha as informações de direitos autorais no início de cada arquivo-fonte e inclua um link para as páginas oficiais no site onde suas visualizações estiverem hospedadas.

    Se você desenvolver algo interessante a partir deste material, fique à vontade para entrar em contato. Será um prazer conhecer seu trabalho — e, eventualmente, incluir um link para ele.
